# 📁 어떻게 작성하나요?

기본적인 git 사용법을 안다는 가정하에 설명합니다. (만약 모르신다면 RyuWoong에게 DM을 주시면 친절하게 설명해드립니다.)
먼저, 본인 닉네임 폴더를 생성 후. 그 안에 MarkDown 파일로 작성하는 것이 기본입니다.

파일명은 아래와 같이 작성합니다.
[1주차] 1장 ~~~~ .MD

내부 내용은 편한대로 작성하시면 됩니다!
